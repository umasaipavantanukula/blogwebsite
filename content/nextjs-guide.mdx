# Getting Started with Next.js

Next.js is a powerful React framework that makes building web applications a breeze.

## Key Features

- Server-side rendering
- Static site generation
- File-based routing
- API routes

## Installation

```bash
npx create-next-app@latest my-app
cd my-app
npm run dev
```

Happy coding!
